# Mini MVC PHP Framework

A lightweight **Laravel-style Mini MVC framework built with PHP** for learning and small projects.
This project demonstrates how the **MVC (Model–View–Controller) architecture** works internally.

---

## 📌 Features

* Simple **MVC architecture**
* Clean **routing system**
* Basic **controller support**
* **View rendering system**
* Lightweight **core framework**
* Easy to understand for beginners

---

## ⚙️ Requirements

* PHP **8.0+**
* Apache / Nginx
* Composer (optional)
* XAMPP / Laragon recommended

---

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/Kzt224/Mini_Mvc.git
```

2. Move the project to your server directory

Example (XAMPP):

```
xampp/htdocs/
```

3. Start Apache

4. Open in browser

```
http://localhost/Mini_Mvc/public
```

---

## 🧠 How MVC Works

### Model

Handles **database logic and data processing**.

### View

Responsible for **UI and frontend display**.

### Controller

Handles **application logic and user requests**.
